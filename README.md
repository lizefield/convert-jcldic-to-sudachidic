## Convert JCLdic to Sudachi dic

[TIS、自然言語処理で企業名認識を行うための辞書「JCLdic」を無償公開](https://www.tis.co.jp/news/2020/tis_news/20200420_1.html)  

無償公開されているJCLdic（MeCabフォーマット）をSudachiのユーザー辞書に変換するサンプルです。  

### Python 仮装環境設定

create  

```
python -m venv .venv
```

activate  

```
source .venv/bin/activate
```

deactivate  

```
deactivate
```
